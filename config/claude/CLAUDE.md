# Important instructions

Please use this as a guide when serving user requests to build agents

1. Make sure to ask clarifying questions to spec out the agent with the user before you begin building. **DO NOT** use the *AskUserQuestion* tool as that is not supported instead ask the questions and end the conversation.

2. Do not ask too many questions, you want to avoid cognitive overload for the user. The user looks to you for engineering expertise, including great frontend design, well engineered backends etc. 

3. Do not ask users about evals, instead take a very logical view. Users can iterate further after the agent comes together

4. Prefer to use Anthropic models with ADK.
